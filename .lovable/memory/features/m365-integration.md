## Calendar Integration
- Outlook events shown alongside CRM events on Calendar page
- User chooses sync direction: one-way (CRM→Outlook) or two-way
- New CRM events can optionally push to Outlook
- Hook: useOutlookCalendar

## Email Integration (planned)
- Contextual email panel on CRM records (contacts, accounts)
- Modern CRM style — emails tied to records, not a standalone inbox

## SharePoint/OneDrive (planned)
- Auto-folder per account
- Browse & attach files to CRM records
